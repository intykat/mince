# API Reference

Welcome to the Mince API reference. This section provides a detailed, technical breakdown of the functions, methods, and objects available in the Mince framework.

For more detailed explanations and best practices, please see the **[Guides](/mince/guide)**.

## Framework
- [**Core API**](./framework): Core functions for loading and accessing modules.
- [**Module API**](./module): Special lifecycle methods for Mince modules.

## Components
- [**Component API**](./component): API for creating and managing components.

## Networking
- [**Event API**](./event): For one-way, real-time communication.
- [**Function API**](./function): For request/response style communication.
- [**State API**](./state): For synchronizing complex data structures.
